# Araba Yarışı Oyunu

Bu proje, **C# Windows Forms** kullanılarak geliştirilmiş basit bir **araba yarışı oyunudur**.  
Oyuncu, kendi arabasını sağa ve sola hareket ettirerek diğer araçlara çarpmadan ilerlemeye çalışır.  
Oyun ilerledikçe puan artar ve hız değişimleriyle oyun daha zor hale gelir.



# Özellikler

- Sağa–Sola hareket: Klavye yön tuşlarıyla kontrol edilir.  
- Çarpışma algılama: Diğer araçlarla temas anında oyun biter.  
- Patlama efekti: Çarpışma anında görsel efekt gösterilir.  
- Arka plan sesi: Oyun başladığında ses efekti çalar.  
- Puan sistemi: Oyuncunun dayanma süresine göre puan artar.  
- Yeniden başlatma butonu: `Oyunu Başlat` tuşu ile oyun sıfırlanır.



# Kullanılan Teknolojiler

- Programlama Dili: C#  
- Arayüz: Windows Forms  
- Kütüphaneler:
  - `System.Media` → Ses dosyası çalma  
  - `System.Drawing` → Görseller  
  - `System.Windows.Forms` → Oyun arayüzü
